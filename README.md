# What_am_I_copying

실시간으로 클립보드를 모니터링하고 텍스트와 이미지 복사 기록을 관리하는 Chrome 확장 프로그램입니다.

---

## ## 주요 기능

* **실시간 클립보드 모니터링**: 텍스트와 이미지를 복사할 때마다 이를 감지하여 히스토리에 저장합니다.
* **히스토리 관리**: 복사한 기록을 최대 100개까지 저장합니다.
* **중복 방지**: 마지막으로 복사한 내용과 동일한 항목은 히스토리에 중복으로 추가하지 않습니다.
* **다양한 관리 기능**:
    * **다시 복사**: 히스토리 목록에서 원하는 항목을 클릭하여 클립보드에 다시 복사할 수 있습니다.
    * **파일로 저장**: 텍스트는 `.txt` 파일로, 이미지는 `.png` 파일로 다운로드할 수 있습니다.
    * **삭제**: 특정 항목을 개별적으로 삭제하거나, 전체 히스토리를 한 번에 비울 수 있습니다.
    * **확대 보기**: 내용이 긴 텍스트나 이미지는 모달(Modal) 창에서 크게 확인할 수 있습니다.
* **즉각적인 UI 반영**: 팝업이 열려 있는 상태에서 새 항목이 복사되어도 목록이 실시간으로 업데이트됩니다.
* **모던한 UI**: VS Code 스타일의 깔끔한 다크 모드 인터페이스를 제공합니다.

## ## 설치 및 사용법

### ### 설치 방법 (로컬에서 설치)

현재 Chrome 웹 스토어에 등록되지 않았으므로, 다음과 같이 '개발자 모드'를 통해 수동으로 설치해야 합니다.

1.  이 GitHub 저장소의 모든 파일을 다운로드하거나 `git clone` 받습니다.
2.  Chrome 브라우저를 열고, 주소창에 `chrome://extensions` 를 입력하여 확장 프로그램 관리 페이지로 이동합니다.
3.  페이지 우측 상단의 **'개발자 모드(Developer mode)'** 스위치를 켭니다.
4.  좌측 상단에 나타난 **'압축 해제된 확장 프로그램을 로드합니다(Load unpacked)'** 버튼을 클릭합니다.
5.  파일 탐색기에서 이 프로젝트를 다운로드한 `What_am_I_copying` 폴더 전체를 선택하고 '확인'을 누릅니다.
6.  설치가 완료되면 브라우저 툴바에 확장 프로그램 아이콘이 나타납니다. (보이지 않으면 툴바의 퍼즐 아이콘을 눌러 고정하세요.)

### ### 사용 방법

1.  브라우저 툴바의 **`What_am_I_copying` 아이콘을 클릭**하여 팝업 창을 엽니다.
2.  **현재 클립보드**: 팝업 최상단에 현재 클립보드에 저장된 내용이 표시됩니다.
3.  **복사 히스토리**: 그 아래로 이전에 복사했던 텍스트와 이미지 목록이 시간순으로 나열됩니다.
4.  **항목 관리**: 각 히스토리 항목에서 다음 작업을 수행할 수 있습니다.
    * **📋 복사**: 해당 항목을 클립보드에 다시 복사합니다.
    * **💾 .txt / 💾 .png**: 해당 항목을 텍스트 파일 또는 이미지 파일로 저장합니다.
    * **🗑️ 삭제**: 해당 항목을 히스토리에서 제거합니다.
    * **(내용 클릭)**: 텍스트나 이미지를 모달 창으로 크게 봅니다.
5.  **전체 관리**:
    * **새로고침**: 현재 클립보드와 히스토리 목록을 수동으로 새로고침합니다.
    * **전체삭제**: 확인 창과 함께 모든 히스토리를 삭제합니다.

## ## 사용된 주요 기술

* **Chrome Manifest V3**
* **Service Worker** (백그라운드 모니터링)
* **Offscreen Document API** (MV3에서 DOM API인 Clipboard 접근)
* **Clipboard API** (`navigator.clipboard.read`, `readText`)
* **Chrome Storage API** (`chrome.storage.local`)
* **Chrome Alarms API** (주기적인 클립보드 체크)
* HTML / CSS / Vanilla JavaScript
